== Description ==


Allow you to install and access full feature of Google Analytics inside your WP Admin Page. It is really simple but powerful

GA can track visitors from all referrers, including search engines, display advertising, pay-per-click networks, e-mail marketing and digital collateral such as links within PDF documents.

[more info](http://mr.hokya.com/analytics-installer/)

== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
